# Translation status

**English ([`README.md`](../../README.md)) is canonical.** This index tracks every
translation and its review tier. It is validated by
[`scripts/check-readme-translations.mjs`](../../scripts/check-readme-translations.mjs)
(run in CI). See [TRANSLATING.md](TRANSLATING.md) to contribute.

- **canonical** — the source of truth.
- **reviewed** — translated and reviewed by a fluent human.
- **machine-assisted** — drafted with machine assistance; **needs human review**.
  Accuracy is not guaranteed; when in doubt, read the canonical English.
